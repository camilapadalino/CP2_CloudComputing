O que é PaaS?

PaaS (Plataforma como Serviço) basicamente é um modelo de cloud onde o provedor oferece toda a infraestrutura pronta, como servidor, sistema operacional e ambiente de execução, para que o desenvolvedor foque apenas no código da aplicação, sem se preocupar com configurações técnicas por baixo.
